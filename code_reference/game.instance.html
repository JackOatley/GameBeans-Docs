GAME.instance

The GAME.instance object contains functions for dealing with instances. For creating them, manipulating them and destroying them.

Contents:

  • create
  • createMoving
  • destroy
  • count
  • mouseOn
  • find
  • findRandom

GAME.instance.create(object, x, y)

Creates a new instance of an object.

Parameters:

  • object - The name, as a string, of the object of which to create an instance of.
  • x - The X position, as a number, to create the instance.
  • y - The Y position, as a number, to create the instance.
// Create the player in the middle of the room. GAME.instance.create(objPlayer, 320, 240);

GAME.instance.createMoving(object, x, y, speed, direction)

Creates a new instance of an object, and starts it moving at speed in a direction.

Parameters:

  • object - The name, as a string, of the object of which to create an instance of.
  • x - The X position, as a number, to create the instance.
  • y - The Y position, as a number, to create the instance.
  • speed - The speed, as a number, at which the instance shall move.
  • direction - The direction in degrees, as a number, in which the instance shall move.
// Create the player in the middle of the room and zoom away! GAME.instance.createMoving(objPlayer, 320, 240, 10, 90);

GAME.instance.destroy(inst)

Destroys the given instance.

Parameters:

  • inst - The instance object to be destroyed. This parameter can also be the this keyword, or the string "other" in the case of a collision event.
// destroy the calling instance GAME.instance.destroy(this); // destroy the other instance in a collision event GAME.instance.destroy(other); // destroy the first found instance of the given object var inst = GAME.instance.find(objSomeObject, 0); GAME.instance.destroy(inst);

GAME.instance.count(object)

Returns the number of instances of the given object that currently exist.

Parameters:

  • object - An object, as a string.
// How many instances are there? var count = GAME.instance.count(objSomeObject); console.log("Number of instances: " + count)

GAME.instance.mouseOn(inst)

Returns true if the mouse is over the given instance.

Parameters:

  • inst - The instance to check.
// Hovering... if (GAME.instance.mouseOn(this)) console.log("hovering...");

GAME.instance.find(object, index)

Finds and returns the Nth instance of an object.

Parameters:

  • object - The object, as a string, to find an instance of.
  • index - The index, as an integer, of the instance.
// Find the first instance of the object and log it's position in the browser console. const inst = GAME.instance.find(objSomeObject, 0); console.log(inst.x, inst.y)

GAME.instance.findRandom(object)

Finds and returns any instance of an object.

Parameters:

  • object - The object, as a string, to find an instance of.
// Find any instance of the object and log it's position in the browser console. var inst = GAME.instance.findRandom(objSomeObject); console.log(inst.x, inst.y)
