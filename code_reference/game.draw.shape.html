GAME.draw.shape

The GAME.draw.shape object contains functions for drawing shapes.

Contents:

  • GAME.draw.shape.line()
  • GAME.draw.shape.ellipse()
  • GAME.draw.shape.rectangle()
  • GAME.draw.shape.healthBar()

GAME.draw.shape.line(x1, y1, x2, y2, options)

Draws a line between the given coordinates.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.
  • options - OPTIONAL parameters.
    • color - The color of the line, defaults to global drawing color.
// Draw a green line from a to b. GAME.draw.shape.line(a.x, a.y, b.x, b.y, { color: "green" });

GAME.draw.shape.ellipse(x, y, xr, yr, options)

Draws elliptical shapes, like ovals and circles.

Parameters:

  • x - The X position.
  • y - The Y position.
  • xr - The X radius.
  • yr - The Y radius.
  • options - OPTIONAL object with advanced parameters.
    • color - The color of the ellipse, defaults to global drawing color.
    • fill - Should the ellipse be filled or stroked.
    • lineWidth - The width of the stroke, if stroke is used.
// Draw a red circle. GAME.draw.shape.ellipse(320, 240, 100, 50, { color: "red" });

GAME.draw.shape.rectangle(x, y, w, h, options)

Draws reactangles.

Parameters:

  • x - The X position.
  • y - The Y position.
  • w - The width.
  • h - The height.
  • options - OPTIONAL object with advanced parameters.
    • color - The color of the rectangle, defaults to global drawing color.
    • fill - Should the ellipse be filled or stroked.
    • lineWidth - The width of the stroke, if stroke is used.
// Draw a blue rectangle. GAME.draw.shape.rectangle(92, 28, 322, 10{ color: "blue" });

GAME.draw.shape.healthBar(x, y, w, h, amount, color, background)

Draws rectangular healthbars.

Parameters:

  • x - The X position.
  • y - The Y position.
  • w - The width.
  • h - The height.
  • amount - The amount of health, out of 100.
  • color - The color to draw the "healthy" part of the bar.
  • background - the color to draw the "damaged" part of the bar.
GAME.draw.shape.healthBar(92, 28, 322, 10, health, "lime", "red");
