GAME.Font(options)

Creates a new tilemap.

Parameters:

  • options - OPTIONAL. You can use any, all or none of these parameters.
    • name - The resource name that the font will have.
    • src - The source of the font (as a url or base64-encoded string).
    • bitmap - An existing bitmap. Don't use this for now, not fully supported.
    • method - Initial drawing method, defaults to "normal", can be set to "bitmap".
    • apply - If true, assigns the font to a @font-face in a style tag.
const myFont = new GB.Font({ name: "fntMyFont", src: source, apply: true });

getCss(font)

Returns a string of CSS code that's needed to be applied to the page to use this font in the DOM or on a canvas.

Parameters:

  • font - The Font resource.
const css = fntMyFont.getCss();

applyCss(font)

Applies the CSS code from GAME.Font.getCss().

Parameters:

  • font - The Font resource.
fntMyFont.applyCss();

convertToBitmapFont(font)

Converts the vector font to a bitmap font.

Parameters:

  • font - The Font resource.
  • options
    • size - Size of the font.
    • map - A string of all the characters to include in the bitmap font.
    • alphaThreshold - Threshold that dertmines which pixels shall be visible.
    • color - An array containing the RGBA channels respectively.
// Our options. var options = { size: 16, map: "ABCabc", alphaThreshold: 150, color: [255, 0, 0, 255] } fntMyFont.convertToBitmapFont(options);

getByName(name)

Returns the font resource of the given name.

Parameters:

  • name - The string name of the font to get.
var font = GAME.Font.getByName("fntMyFont");
