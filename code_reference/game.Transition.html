GAME.Transition

GAME.Transition is a class object, and so is used as a constructor. A transition is split into 2 phases; "in" and "out". At the end of the "in" phase, if the transition has a callback it will be executed, then the "out" phase begins. At the end of the "out" phase, the transition is destroyed.

Contents:

  • GAME.Tilemap()

GAME.Transition(options)

Starts a transition.

Parameters:

  • options - OPTIONAL. You can use any, all or none of these parameters.
    • type - The type of the transition.
    • time - The time, in game steps, that the transition should last in each phase.
    • color - The color passed into the rendering function.
    • callback - The function to execute at the mid-pont of the transition.
    • prefab - A previously created object that overwrites provided parameters.
// Go to another room new GAME.Transition({ type: "fade", time: 60, callback: function() { GAME.room.enter("rmLevel12"); } });
