GAME.sound

The GAME.sound object contains functions for dealing with sounds.

Contents:

  • GAME.sound.play
  • GAME.sound.stop
  • GAME.sound.get

GAME.sound.play(snd, options)

Plays a sound. By default the sound will play just once, using the options object lets you set a sound to loop as well as other features!

Parameters:

  • snd - The name or object of the sound to play.
  • options - OPTIONAL. An object containing advanced options.
    • loop - Whether the sound should loop when it ends.
    • onEnd - A function to execute when the sound ends.

Returns:

Audio Node - The audio node about to be played. HTML5 only allows a single audio node to be played at a time, due to this, GameBeans creates new nodes as needed. As such, the node returned may not always be the same one.

// 1, 2... 1, 2, 3, 4! GAME.sound.play(sndSickBeat, { loop: true }); // Play an intro, leading to a loop. GAME.sound.play(sndIntro, { onEnd: () => GAME.sound.play(sndMainLoop, { loop: true }) });

GAME.sound.stop(snd)

Stops all instance of the given sound.

Parameters:

  • snd - The name or object of the sound to stop.

Returns:

Undefined.

// start playung music GAME.sound.play(sndMusic); // in 10 seconds, stop the music. setTimeout(() => { GAME.sound.stop(sndMusic); }, 10 * 1000 );

GAME.sound.get(snd)

Returns the sound resource object from the given name. This object grants direct access to the audio nodes creates when playing the sound.

Parameters:

  • snd - The name of the sound to get the object of.

Returns:

Object - sound resource.

// Start playung music. const soundObject = GAME.sound.get(sndMusic);
