GAME.Primitive

GAME.Primitive is a class object, and so is used as a constructor. A Primitive is a shape made up of a series of points (vertexes), that contain position and UV data.

Contents:

  • GAME.Primitive

GAME.Primitive(options)

Creates a Primitive instance.

Parameters:

  • options - OPTIONAL.
    • points - An array of points that make up the primitive.
    • texture - A Sprite resource as an object or string name.
    • color - The color (in absense of a texture) of the primitive.
    • format - The format of the array of points.
    • modifier - A modifier function that can be used to effect the points during drawing.
    • x - X position of the whole primitive.
    • y - Y position of the whole primitive.
// Create primitive from data this.shape = new GAME.Primitive({ texture: "sprTexture", format: "quads", points: points, modifier: modifier });
