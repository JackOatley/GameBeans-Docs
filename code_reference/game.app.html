GAME.App

The GAME.app object contains functions for dealing with the application as a whole.

Contents:

  • setCaption()
  • setCursor()
  • restart()

GAME.App.setCaption(name)

Sets the cursor, while mouse is over the game in the browser.

Parameters:

  • name - The name to appear in the window/browser tab.
GAME.App.setCaption("My Game");

GAME.App.setCursor(cursor)

Sets the cursor, while mouse is over the game in the browser. See MDN Web Docs for a full list of options.

Parameters:

  • cursor - The name of the cursor to use.
GAME.App.setCursor("pointer");

GAME.App.restart(cursor)

Restarts the game.

GAME.App.restart();
