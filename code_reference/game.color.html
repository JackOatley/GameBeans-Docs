GAME.color

The GAME.color object contains functions for dealing with colors.

Contents:

  • GAME.color.hexToRgb()
  • GAME.color.componentToHex()
  • GAME.color.rgbToHex()

GAME.color.hexToRgb(hex)

Takes a hex value represented as a string and returns an object contained the separated RGB components. A hex value would be as such: "#FF00FF". Returns null if the value provided is not valid.

Parameters:

  • hex - String represented hex vlaue.
/* The following code returns an object the looks like this: { r: 255, g: 0, b: 255 } */ var colorAsRgb = GAME.color.hexToRgb( "#FF00FF" );

GAME.color.componentToHex(c)

Takes a color component; R, G or B, and returns its hex value without the hash, such as "00" or "FF".

Parameters:

  • hex - String represented hex value.
// Where's the hexit? var red = GAME.color.componentToHex( 255 ); // red = "FF"

GAME.color.rgbToHex(r, g, b)

Takes a paremeter for each component of a color and returns the complete hex value as a string.

Parameters:

  • r - The red component.
  • g - The green component.
  • b - The blue component.
// convert components to a single hex value var hex = GAME.color.rgbToHex( 0, 128, 255 ); // hex = "#0080FF"
