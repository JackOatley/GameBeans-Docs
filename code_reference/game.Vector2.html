GAME.Vector2

The Vector2 class represents a 2D Vector; a pair of values "x" and "y".
2D Vectors can represets 2D positions, angles, velocities, etc.

Creates a new Vector2 with the given values.

Parameters:

  • x - X component of the vector.
  • y - Y component of the vector.
