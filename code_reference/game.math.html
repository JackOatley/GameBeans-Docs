GAME.Maths

The GAME.Maths object contains constants and functions for dealing with math, some functions are just copied from the JavaScript Math object but a lot are unique. Rationale behind using Maths over Math is that Math is already used by JavaScript.

GAME.Maths.angleDifference(angle1, angle2)

Returns the difference, in degrees, between 2 angles. This value can range from -180 to 180.

Parameters:

  • angle1 - The first angle.
  • angle2 - The second angle.

Gradually turn to face an enemy.

const direction = GAME.Maths.pointDirection(this.x, this.y, enemy.x, enemy.y); const facing = Math.sign(GAME.Maths.angleDifference(this.rotation, direction)); this.rotation += facing;

GAME.Maths.chance(probability)

Returns true with the given probability.

Parameters:

  • probability - The chance to return true: between 0 > 1.
if (GAME.Maths.chance(0.5)) { console.log("You had a 50% chance of seeing this!"); }

GAME.Maths.choose(...params)

This function takes an arbitrary number of parameters and returns just one of them, chosen at random.

const food = GAME.Maths.choose("burgers", "pie", "fish", "kale"); console.log(food); // Will print one of the above.

GAME.Maths.clamp(number, min, max)

Returns the given number clamped between a min and max value.

Parameters:

  • number - Any number.
  • min - The minimum value allowed to return.
  • mac - The maximum value allowed to return.
GAME.Maths.clamp(4, 0, 10); // Returns 4. GAME.Maths.clamp(-5, 0, 10); // Returns 0. GAME.Maths.clamp(15, 0, 10); // Returns 10.

GAME.Maths.lengthDir(len, dir)
GAME.Maths.lengthDirX(len, dir)
GAME.Maths.lengthDirY(len, dir)

These functions return the X and Y positions, from 0, of a point at the specified distance away in the given direction.

Parameters:

  • len - The length/distance from 0 of the new point.
  • dir - The direction in which the "len" reaches out, in degrees.

Some simple movement code.

this.x += GAME.Maths.lengthDirX(2, this.rotation); this.y += GAME.Maths.lengthDirY(2, this.rotation);

Same as above using to function that returns an array of both coordiantes.

const [x, y] = GAME.Maths.lengthDir(2, this.rotation); this.x += x; this.y += y;

GAME.Maths.pointDistance(x1, y1, x2, y2)

Returns the distance between 2 points.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.
const distance = GAME.Maths.pointDistance(earth.x, earth.y, moon.x, moon.y); // It's 384,400 km, btw.

GAME.Maths.pointDirection(x1, y1, x2, y2)

Returns the direction, in degrees, between 2 points. As a reference; 0 is right, 90 is down, 180 is left and 270 is up.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.

Return the direction from the player to a quest.

const direction = GAME.Maths.pointDirection(player.x, player.y, quest.x, quest.y);

GAME.Maths.randomInt(min, max)

Returns a random integer within the given range. Both values in this function are inclusive, so if you enter 5 and 9 as min and max respectively, then both those values are possible return values.

Parameters:

  • min - The min value in the range.
  • max - The max value in the range.

Pick a number between 0 and 30.

let number = GAME.Maths.randomInt(0, 30);

GAME.Maths.RADTODEG
GAME.Maths.DEGTORAD

Variables used to convert between radians and degrees. The value of RADTODEG is (180 / PI) and the value of DEGTORAD is (PI / 180).

Convert radians to degrees.

const radians = 1.5708; const degrees = radians * GAME.Maths.RADTODEG; // 90

Convert degrees to radians.

const degrees = 90; const radians = degrees * GAME.Maths.DEGTORAD; // 1.5708
