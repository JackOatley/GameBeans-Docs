GAME.Room

GAME.Room( name, width, height )

Create a new room.

Parameters:

  • name - The name of the new room. This should be unique.
  • width - The width of the new room.
  • height - The height of the new room.
// Create a new room var newRoom = new GAME.Room("rmNewRoom", 640, 480);

GAME.Room.create(name, width, height)

See Constructor for details.

// Create a new room. const newRoom = GAME.Room.create("rmNewRoom", 640, 480);

GAME.Room.enter(room, options)

Goes to the given room.

Parameters:

  • room - The name or object of the room to go to.
  • options - OPTIONAL
    • transition - Automatically creates a transition and handles the room change callback. Takes an object with properties matching that of GAME.Transition.
// Go to another room. GAME.Room.enter(rmLevel2); // or GAME.Room.enter("rmLevel2"); // As method rmLevel2.enter(); // With transition GAME.Room.enter(rmLevel2, { transition: { type: "fade", time: 60 } });

GAME.Room.next()

Goes to the next room, meaning the room after placed after the current one in the resource tree.

// Go to the next room. GAME.Room.next();

GAME.Room.previous()

Goes to the previous room, meaning the room after placed before the currentone in the resource tree.

// Go to the previous room GAME.Room.previous();

GAME.Room.setBackground(room, sprite)

Sets the sprite to use as the background from the given room.

Parameters:

  • room - The room to change the background of.
  • sprite - The sprite to use as the background.
// Change the background. const room = GAME.Room.current; GAME.Room.setBackground(room, sprBackground2);

GAME.Room.addInstance(room, object, x, y)

Add an instance of the given object to a room. Note that this does not add it immedietly, like GAME.instance.create() does, instead it adds it to the room so that the every subsequent time you enter that room the instance will be there.

Parameters:

  • room - The room.
  • object - The object to create an instance of.
  • x - The X position to create the instance.
  • y - The Y position to create the instance.
// Create a new room, add an instance, then enter the room var newRoom = GAME.Room.create("rmNewRoom", 640, 480); GAME.Room.addInstance(newRoom, objPlayer, 320, 240); GAME.Room.enter(newRoom);

GAME.Room.draw(room)

Draws the given room. Currently, this only means it's background. This is done automatically for the room you're in. But for whatever reason if you want to re-draw it or draw a different room in the current one, then you can with this function.

Parameters:

  • room - The room to draw.
// Draw the room. GAME.Room.draw(rmNewRoom);

GAME.Room.get(room)

Returns the resource object of the current room by it's string name. With this object you can access various properties of the room, and even change them directly. Care needs to be taken when dealing with resource objects directly.

Parameters:

  • room - The name of the room to get.
// Get current room size var room = GAME.Room.get("rmLevel2"); var width = room.width; var height = room.height;
