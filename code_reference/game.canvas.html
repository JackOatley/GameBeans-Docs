GAME.Canvas

The Canvas object contains functions for dealing with canvases. Canvases, in HTML5, are what we use to render graphics. GameBeans' drawing functions draw to a canvas on the webpage, and this is how you see your game.

Constructors

Canvas(options)

Creates and returns a new Canvas.

Parameters:

  • options - OPTIONAL. An object containing advanced parameters.
    • width - The width of the canvas. Defaults to 300.
    • height - The height of the canvas. Defaults to 150.
const myCanvas = new GAME.Canvas({ width: 640, height: 480 });

Public Methods

fill(color)

Fill the Canvas to the given color. Note that this is done automatically for the main canvas at the start of the Draw Event based on the current Room background color.

myCanvas.clear("red");

clear()

Clears the Canvas.

myCanvas.clear();

Static Properties

main

Returns the canvas that is currently set to be GameBeans' main canvas (the one on the web page.)

const mainCanvas = GAME.Canvas.main;

Static Methods

setMain()

Sets the main rendering Canvas to the given Canvas. This does not change the Canvas displayed on the webpage (see "unsetMain").

GAME.Canvas.setMain(myCanvas);

unsetMain()

Removes the main Canvas that is currently being used on the webpage. Calling "setMain" after calling this method will add the given Canvas to the webpage.

GAME.Canvas.unsetMain();
