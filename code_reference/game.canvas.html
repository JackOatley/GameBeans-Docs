GAME.canvas

The GAME.canvas object contains functions for dealing with canvases. Canvases, in HTML5, are what we use to render graphics. GameBeans' drawing functions draw to a canvas on the webpage, and this is how you see your game.

Contents:

  • create()
  • getMain()
  • clear()

GAME.canvas.create(options)

Creates and returns new canvas.

Parameters:

  • options - OPTIONAL. An object containing advanced parameters.
    • width - The width of the canvas. Defaults to 300.
    • height - The height of the canvas. Defaults to 150.
// create a new canvas GAME.canvas.create({ width: 640, height: 480 });

GAME.canvas.getMain()

Returns the canvas that is currently set to be GameBeans' main canvas (the one on the web page.)

// get main canvas var mainCanvas = GAME.canvas.getMain();

GAME.canvas.clear(canvas, color)

Clears the given canvas to the given color. This is done automatically for the main canvas at the start of the Draw Event.

// clear the main canvas to red var mainCanvas = GAME.canvas.getMain(); GAME.canvas.clear(mainCanvas, "red");
