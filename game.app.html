GAME.app

The GAME.app object contains functions for dealing with the application as a whole.

Contents:

  • GAME.app.setCursor()

GAME.app.setCursor( cursor )

Sets the cursor, while mouse is over the game in the browser.

Parameters:

  • cursor - The name of the cursor to use.
// don't point at me! GAME.app.setCursor("pointer");
