GAME.draw.shape

The GAME.draw.shape object contains functions for drawing shapes.

GAME.draw.shape.line( x1, y1, x2, y2, options )

Draws a line between the given coordinates.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.
  • options - OPTIONAL parameters.
    • color - The color of the line, defaults to global drawing color.

Example

// draw a red line from a to b GAME.draw.shape.line( z.x, z.y, b.x, b.y, { color: "#FF0000" });

GAME.draw.shape.ellipse( x, y, xr, yr, options )

Draws elliptical shapes, like ovals and circles.

Parameters:

  • x - The X position.
  • y - The Y position.
  • xr - The X radius.
  • yr - The Y radius.
  • options - OPTIONAL object with advanced parameters.
    • color - The color of the ellipse, defaults to global drawing color.
    • fill - Should the ellipse be filled or stroked.
    • lineWidth - The width of the stroke, if stroke is used.
    • healthbar - OPTIONAL object for using the shape as a healthbar.
      • amount - Fraction representing the amount the healthbar should be filled, with 0 being empty and 1 being full.
      • color - The main color, defaults to green.
      • background - The background color, defaults to red.

Example

// draw a healthbar GAME.draw.shape.ellipse( 320, 240, 100, 50, { healthbar: { amount: this.health / this.maxHealth, color: "#FFFFFF", background: "#000000" } } );

GAME.draw.shape.rectangle( x, y, w, h, options )

Draws elliptical shapes, like ovals and circles.

Parameters:

  • x - The X position.
  • y - The Y position.
  • w - The width.
  • h - The height.
  • options - OPTIONAL object with advanced parameters.
    • color - The color of the ellipse, defaults to global drawing color.
    • fill - Should the ellipse be filled or stroked.
    • lineWidth - The width of the stroke, if stroke is used.
    • healthbar - OPTIONAL object for using the shape as a healthbar.
      • amount - Fraction representing the amount the healthbar should be filled, with 0 being empty and 1 being full.
      • color - The main color, defaults to green.
      • background - The background color, defaults to red.

Example

// draw a healthbar GAME.draw.shape.rectangle( 92, 28, 322, 10, { healthbar: { amount: this.health / this.maxHealth, color: "#FFFFFF", background: "#000000" } } );
