Behaviours

Behaviours encapsulate common functionality, allowing you to assign such to an object without having to "reinvent the wheel" each time.

Game

4-Way Movement

Give the object 4 way movement, either through Arrow Keys or WASD.
The object will not be able to move diagonally even if holding up and right keys, for example.

8-Way Movement

Give the object 8 way movement, either through Arrow Keys or WASD. The object will move diagonally when a vertical and horizonatal key are held simultaneously.

Bound To Room

Bind the objeect to the room, preventing it from leaving the room's bounds. If the object tries to move move out of the romo bounds then it will be placed back enough to be touching the room bounds. This happens during the End Step.
