GameObjects

GameObjects are the highest level form of control you have for creating games in GameBeans. They can control and help manage anything. A GameObject can be the player character, a pickup, an enemy, a HUD element, or something invisible that handles some aspect of your game such as enemy spawning, etc.

GameObjects can be given a Sprite that is displayed automatically when placed in a room. A GameObject placed in a room is known as an "Instance", as in "an instance of a GameObject". You can have 1 or many Instances of a single GameObject. Instances inherit everything from the GameObject they are an Instance of, but act independently after creation.

Selecting a GameObject from the resource tree opens it up in the GameObject Editor. Here's you can set it's Sprite and various other properties. Behaviours, Events and Actions can also be assigned to the GameObject here, they are found in the Context menu on the left.

Parameters:

All GameObject instances have the following built in properties. These can be set with the "Set Variable" action, or in code using the "this" keyword.

  • sprite - The GameObject's sprite.
  • visible - Whether instances of this GameObject start visible.
  • depth - Determines drawing order.
  • solid - NA. Defines this object as solid, for use in collisions.
  • scaleX - Horizontal scaling factor of the sprite.
  • scaleY - Vertical scaling factor of the sprite.
  • rotation - Angle of the sprite.
  • terminal - NA. The maximum speed allowed by gravity.
  • gravity - The gravity gain.
  • gravityDirection - Direction to which gravity pulls the instance.
  • speed - Speed that the instance is moving.
  • direction - Direction the instance is moving in.
  • speedX - Horizontal speed.
  • speedY - Vertical speed.
  • boxTop - Y position for the top of the bounding box.
  • boxBottom - Y position for the bottom of the bounding box.
  • boxLeft - X position for the left of the bounding box.
  • boxRight - X position for the right of the bounding box.
