Objects

Objects are the highest level form of control you have for creating games in GameBeans. They can control and help manage anything. An Object can be the player character, a pickup, an enemy, a HUD element, or something invisible that handles some aspect of your game such as enemy spawning, etc.

Objects can be given a Sprite that is displayed automatically when placed in a room. An Object placed in a room is known as an "Instance", as in "an instance of an object". You can have 1 or many Instances of a single Object. Instances inherit everything from the Object they are an Instance of, but act independently after creation.

Selecting an Object from the resource tree opens it up in the Object Editor. Here's you can set it's Sprite and various other properties. Events and Actions can also be assigned to the Object here, they are found in the Context menu on the left.

Parameters:

All objects have the following built in properties. These can be set with the "Set Variable" action, or in code using the "this" keyword.

  • sprite - The object's sprite.
  • visible - Whether the object starts visible.
  • depth - Determines drawing order.
  • solid - NA. Defines this object as solid, for use in collisions.
  • scaleX - Horizontal scaling factor of the sprite.
  • scaleY - Vertical scaling factor of the sprite.
  • rotation - Angle of the sprite.
  • terminal - NA. The maximum speed allowed by gravity.
  • gravity - The gravity gain.
  • gravityDirection - Direction to which gravity pulls.
  • speed - Speed the object is moving.
  • direction - Direction the object is moving in.
  • speedX - Horizontal speed.
  • speedY - Vertical speed.
  • boxTop - Y position for the top of the bounding box.
  • boxBottom - Y position for the bottom of the bounding box.
  • boxLeft - X position for the left of the bounding box.
  • boxRight - X position for the right of the bounding box.
