Events

Events are a group of actions within an Object that execute at a particular time. Events can range from game loop moments like Step and Draw, to inputs and collisions between Objects.

Events:

These are the events, in their logical categories, currently available.

  • Game
    • Create - Executed when a new instance of the Object is created.
    • Step - Executed within the game logic loop, by default this should execute 60 times a second.
    • Draw - Used for drawing. Executed when a new frame is about to be rendered, as such this event can run a variable number of times a second so it should not be used for game logic!
    • Collision - Executed when the current instance collides with another instance of a given a Object.
    • Room Enter - Executed after the game enters a new room.
    • Room Leave - Executed before the game leaves the current room.
  • Input
    • Keyboard - Executed when the given Keyboard key is in the given state.
    • Mouse - Executed when the given Mouse button is in the given state.
  • App
    • Resize - Executed when the window/browser resizes.
