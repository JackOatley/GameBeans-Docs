GAME.draw.transform

The GAME.draw.transform object contains functions for setting the scaling, rotating and translating of all subsequent drawing.

GAME.draw.transform.scale( sX, sY )

Adds the given horizontal and vertical scaling to the transformation.

Parameters:

  • sX - The horizontal scaling.
  • sY - The vertical scaling.

Example

// draw the calling instance, bigger GAME.draw.transform.scale( 2, 2 ); GAME.draw.self( this );

GAME.draw.transform.rotate( rot )

Adds the rotation to the transformation.

Parameters:

  • rot - The rotation, in degrees, to add.

Example

// draw the calling instance, upside down GAME.draw.transform.rotate( 180 ); GAME.draw.self( this );

GAME.draw.transform.translate( x, y )

Adds the given x and y positions to the transformation.

Parameters:

  • x - The X position to add.
  • y - The Y position to add.

Example

// draw the calling instance, somewhere else GAME.draw.transform.translate( 100, 100 ); GAME.draw.self( this );

GAME.draw.transform.setDefault( options )

Sets the default transformation. This will be what the Draw Event sets the transform to when it resets before rendering.

Parameters:

  • options - OPTIONAL parameters.
    • moveX - The X translation.
    • moveY - The Y translation.
    • scaleY - The Y scaling.
    • skewX - The X skew.
    • skewY - The Y skew.
