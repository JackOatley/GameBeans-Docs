GAME.Vector2(x, y)

Creates a new Vector2.

Parameters:

  • x - X component of the vector.
  • y - Y component of the vector.

Example

var point = new GAME.Vector2(25, 50);

GAME.Vector2.set(vec, x, y)

Sets the values of the given vector.

Parameters:

  • vec - The Vector2.
  • x - X component of the vector.
  • y - Y component of the vector.

Example

GAME.Vector2.set(point, 12, 34); // or... point.set(12, 34);

GAME.Vector2.copy(vec1, vec2)

Copies the values of vec2 into vec1.

Parameters:

  • vec1 - The vector to the receive the copy.
  • vec2 - THe vector to have it's values copied.

Example

GAME.Vector2.copy(vec1, vec2); // or... vec1.copy(vec2);

GAME.Vector2.add(vec1, vec2)

Adds the values of vec2 to the values of vec1.

Parameters:

  • vec1 - The first vector.
  • vec2 - The second vector.

Example

GAME.Vector2.add(vec1, vec2); // or... vec1.add(vec2);

GAME.Vector2.mul(vec1, vec2)

Multiplies the values of vec1 by the values of vec2.

Parameters:

  • vec1 - The first vector.
  • vec2 - The second vector.

Example

GAME.Vector2.mul(vec1, vec2); // or... vec1.mul(vec2);

GAME.Vector2.div(vec1, vec2)

Divides the values of vec1 by the values of vec2.

Parameters:

  • vec1 - The first vector.
  • vec2 - The second vector.

Example

GAME.Vector2.div(vec1, vec2); // or... vec1.div(vec2);
