GAME.math

The GAME.math object contains constants and functions for dealing with math.

GAME.math.randomInt( min, max )

Returns a random integer within the given range. Both values in this function are inclusive, so if you enter 5 and 9 as min and max respectively, then both those values are possible return values.

Parameters:

  • min - The min value in the range.
  • max - The max value in the range.

Example

// pick a number between 0 and 30 var number = GAME.math.randomInt( 0, 30 );

GAME.math.choose( ...params )

This function takes an arbitrary number of parameters and returns just one of them, chosen at random.

Example

// what's for tea? var tea = GAME.math.choose( "burgers", "pie", "fish", "kale", "friendly bacteria" ); if ( tea === "kale" || tea === "friendly bacteria" ) { console.error( "ERROR: This stuff ain't edible!" ); }

GAME.math.lengthDirX( len, dir )
GAME.math.lengthDirY( len, dir )

These functions return the X and Y positions respectively, from 0, of a point at the specified distance away in the given direction.

Parameters:

  • len - The length/distance from 0 of the new point.
  • dir - The direction in which the "len" reaches out, in degrees.

Example

// movement this.x += GAME.math.lengthDirX( 2, this.rotation ); this.y += GAME.math.lengthDirY( 2, this.rotation );

GAME.math.pointDistance( x1, y1, x2, y2 )

Returns the distance between 2 points.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.

Example

// how far away is the moon? var distance = GAME.math.pointDistance( earth.x, earth.y, moon.x, moon.y ); // It's 384,400 km, btw

GAME.math.pointDirection( x1, y1, x2, y2 )

Returns the direction, in degrees, between 2 points. As a reference; 0 is right, 90 is down, 180 is left and 270 is up.

Parameters:

  • x1 - The X position of the first point.
  • y1 - The Y position of the first point.
  • x2 - The X position of the second point.
  • y2 - The Y position of the second point.

Example

// in which direction is my quest? those 20 arbitrary badguys won't kill themselves! var direction = GAME.math.pointDirection( player.x, player.y, quest.x, quest.y );

GAME.math.angleDifference( angle1, angle2 )

Returns the difference, in degrees, between 2 angles. This value can range from -180 to 180.

Parameters:

  • angle1 - The first angle.
  • angle2 - The second angle.

Example

// turn and face the enemy! var direction = GAME.math.pointDirection( this.x, this.y, enemy.x, enemy.y ); var facing = Math.sign( GAME.math.angleDifference( this.rotation, direction ) ); this.rotation += facing;

GAME.math.PI

Variable that has the value of pi. Same as Math.PI.

Example

// i like pi var pi = GAME.math.PI; var morePi = Math.PI;

GAME.math.RADTODEG
GAME.math.DEGTORAD

Variables used to convert between radians and degrees. The value of RADTODEG is (180 / PI) and the value of DEGTORAD is (PI / 180).

Example

// convert radians to degrees var radians = 1.5708; var degrees = radians * GAME.math.RADTODEG; // 90 // convert degrees to radians var degrees = 90; var radians = degrees * GAME.math.DEGTORAD; // 1.5708

GAME.math.cos
GAME.math.sin
GAME.math.sign

Functions, all the same as their counterparts in the Math library.
